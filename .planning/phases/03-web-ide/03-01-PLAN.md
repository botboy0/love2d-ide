5. Create `ide/routes/files.js`: Express Router with three endpoints:
   - `GET /` â€” Returns recursive directory tree of `config.projectPath` as JSON. Structure: `{ name, type: 'file'|'directory', children? }`. Skip `node_modules`, `.git`, hidden files.
   - `GET /:path(*)` â€” Returns file content as plain text. Path is relative to project root. Validate path stays within project directory (path traversal prevention).
   - `PUT /:path(*)` â€” Writes file content from request body (plain text). Same path validation.

**Important:** The server runs on Windows. All file paths in config.json and route handlers use Windows backslash paths. Use `path.join()` and `path.resolve()` â€” Node.js `path` module handles OS-appropriate separators automatically.
